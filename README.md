# img2binary

Tool to convert an image into 0 and 1. A black and white picture is suggested. 0 is for black and 1 is for white.

# Requirements

- Python 3
- OpenCV

If you donâ€™t have OpenCV installed, install it via pip:

    > pip install opencv-python

# Usage

Given `smile.png` like this:
    
![smile](images/smile.png)

Below is the command to generate binary data into `smile.txt`:

    > python img2binary.py smile.png smile.txt
